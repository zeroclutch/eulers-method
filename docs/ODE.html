Class: ODE

ODE(options)

This program numerically solves a system of N first-order differential equations or a single higher-order ODE. It uses Euler's method.

Constructor

new ODE(options)

Constructs a new ODE in the form y(t)
Parameters:
Name Type Description
options Object The options fore this ODE
Properties
Name Type Description
h number The h value, lower is more accurate
t_o number The initial t value
initialConditions Array.<string> An array of initial conditions
equations Array.<string> An array of equations
iterations number The number of iterations to perform. The final t value t_f is equal to t_o + h * iterations.

Example
const equation = new ODE({
    equations: ["y'''=y''*y'*y+t"],
    initialConditions: ["y'''(0)=0", "y''(0)=0", "y'(0)=0", "y(0)=0"],
    h: 0.1,
    t_o: 0,
    iterations: 20,
})
